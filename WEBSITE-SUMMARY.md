## 🎨 Customization Guide

### **Change Colors**
Edit Tailwind config in each file:
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#YOUR_COLOR',
                secondary: '#YOUR_COLOR',
            }
        }
    }
}
```

### **Update Content**
1. Open HTML file
2. Find section (clearly commented)
3. Replace text/images
4. Save and refresh

### **Add Sections**
1. Copy existing section
2. Modify content
3. Add AOS animation:
   ```html
   <div data-aos="fade-up">
   ```
